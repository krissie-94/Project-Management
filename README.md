Enrich Non-Governmental Organization – Website
Redesign Project Plan & Gantt Chart
Project Title: Enrich NGO Website Redesign
Project Duration: 10 Weeks
Project Goal: Redesign Enrich NGO’s website to improve accessibility, increase donor engagement,
and streamline online program registration. The redesigned site aims to increase donation conversions
by 20% and improve overall user experience.
Project Scope
In Scope:
- Redesign of homepage, About page, Donate page, Program pages, and Contact section
- Creation of new UI/UX mockups following accessibility standards (WCAG 2.1)
- Integration of improved donation flow (simplified forms and confirmation emails)
- SEO improvements (metadata, alt-text, speed optimization)
- User testing with donors, volunteers, and beneficiaries
- Migration of content to new design templates
Out of Scope:
- Rebranding or new logo creation
- Major backend CMS changes
- New mobile app development
Deliverables
- Website requirements document
- UI/UX wireframes and final design
- Updated responsive web pages
- Donation workflow redesign
- SEO and accessibility audit report
- User testing report
- Final deployed website & project closure summary
Milestones
- Week 2: Requirements finalized
- Week 3: UI/UX mockups approved by NGO board
- Week 6: Development completed
- Week 7: Accessibility and SEO testing completed
- Week 8: User testing completed
- Week 10: Website launch and project closure
